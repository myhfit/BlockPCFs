# Introduce
BlockPCFs is sub project for BlockP Projects. 

# Features  
>Add read/write support to dsv files like csv/tsv  
>Add format register for bmp/gif/jpeg/png/m3u/m3u8  
>Add transformer for byte[]>Base64  
>Add transformer for byte[]>Hex String  
>Add transformer for Image>byte[]  
>Add transformer for JSON String>Object
